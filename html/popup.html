modal과 popup의 차이

modal

modal은 시작(진입 컨퍼넌트)하여 끝(OK, Save, Done...)을 낼 수 있으며, 모달에서 다시 바닥(main)로 이동

특직 : Dim 처리

  • Dialog
    • Alert
    • Confirmation
    • Full-screen
  • Navigation Drawer
    • Side
    • Bottom
  • Sheet
    • Side
    • Bottom

non-modal

popup이라 호칭해야 하며 부모 서비스와 함꼐 사용할 수 있음

특징 : Dim 처리 없음

  • Snacbar
  • Banner
  • Menu
  • Tooltip
