modal과 popup

modal과 popup 사용

modal

modal은 시작(진입 컨퍼넌트)하여 끝(OK, Save, Done...)을 낼 수 있으며, 모달에서 다시 바닥(main)로 이동

특징 : Dim 처리

popup

non-modal이라 호칭하는 것이 더 적합하며 부모 서비스와 함꼐 사용할 수 있음

바닥(main)에 정보를 그룹핑 하여 보여주는 기능

특징 : Dim 처리 없음

modal과 popup의 종류

modal

  • Dialog
    • Alert
    • Confirmation
    • Full-screen
  • Navigation Drawer
    • Side
    • Bottom
  • Sheet
    • Side
    • Bottom

non-modal

  • Snacbar
  • Banner
  • Menu
  • Tooltip
